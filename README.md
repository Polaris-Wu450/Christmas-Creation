This is a magical, interactive 3D Christmas tree experience built with React and Three.js.

## Tech Stack

- **React Three Fiber**: For declarative 3D scenes.
- **Three.js**: Core 3D rendering engine.
- **MediaPipe**: For real-time hand tracking and gesture recognition.
- **Post-processing**: Bloom and Vignette effects for that magical glow.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  Open your browser (usually at `http://localhost:5173`) and enjoy the magic!
